                        PROCHAINES ÉTAPES
═══════════════════════════════════════════════════════════════════

1. LIRE README_BELLEYA_MIGRATION.md
   → Point d'entrée principal
   → Choisir son parcours (Quickstart ou Complet)

2. CRÉER LE PROJET BELLEYA
   → Sur supabase.com
   → Récupérer URL et anon key

3. EXÉCUTER LES SCRIPTS SQL
   → belleya_schema_complete.sql dans SQL Editor
   → Créer les 4 buckets Storage
   → belleya_storage_policies.sql pour les policies

4. CONFIGURER L'ENVIRONNEMENT
   → Copier .env.belleya.example vers .env
   → Remplacer par vos vraies credentials

5. TESTER EN LOCAL
   → npm run dev
   → Vérifier le log de démarrage
   → S'inscrire et tester

6. EXÉCUTER LES TESTS
   → Suivre BELLEYA_TEST_CHECKLIST.md
   → Valider les 23 tests

7. DÉPLOYER EN PRODUCTION
   → npm run build
   → Configurer variables sur Netlify/Vercel
   → Déployer
