# Système de Génération de Contenu Intelligent

## Vue d'ensemble

Belleya intègre un système de génération de contenu IA ultra-stratégique qui crée des scripts **humains, experts et engageants**, parfaitement alignés avec le thème, le métier et l'objectif marketing.

---

## Principes fondamentaux

### ❌ Ce qu'on ne fait JAMAIS

1. **Répéter mécaniquement le thème** à chaque slide
2. **Écrire des validations robotiques** type "100% conforme"
3. **Créer du contenu générique** ou flou
4. **Dériver vers un autre sujet**
5. **Utiliser un ton IA** détectable

### ✅ Ce qu'on fait TOUJOURS

1. **Rester strictement aligné** avec le thème fourni
2. **Créer un contenu humain** et naturel
3. **Apporter une vraie valeur** (explication concrète, mécanisme réel, exemple précis)
4. **Intégrer le métier** dans les solutions et l'angle
5. **Optimiser rétention + conversion**

---

## Structure obligatoire (7 étapes)

### 1️⃣ HOOK - Pattern Interrupt Puissant

**Objectif** : Arrêter le scroll en 3 secondes

**Caractéristiques** :
- Phrase courte et percutante
- Émotion ou contradiction
- Curiosité immédiate
- Optimisé pour la rétention

**Exemple** :
```
"Si tu te ronges les ongles… ce n'est pas un manque de volonté."

C'est un mécanisme.
```

---

### 2️⃣ IDENTIFICATION

**Objectif** : Créer la reconnaissance immédiate

**Caractéristiques** :
- Situation réelle vécue par l'audience
- Pensées internes du client
- Micro storytelling si pertinent

**Exemple** :
```
Tu t'es déjà dit :

"C'est la dernière fois."
"Demain j'arrête."

Mais au moindre stress…
Tes doigts reviennent à ta bouche.

Sans même que tu t'en rendes compte.
```

---

### 3️⃣ MÉCANISME / CAUSE RÉELLE

**Objectif** : Apporter une vraie valeur éducative

**Caractéristiques** :
- Explication crédible et experte
- Insight que personne ne donne
- Pas de banalités
- Valeur réelle immédiate

**Exemple** :
```
Se ronger les ongles n'est pas une habitude "bête".

C'est souvent :
• Un réflexe d'anxiété
• Une décharge émotionnelle
• Un besoin de contrôle

Le problème n'est pas tes ongles.
C'est le déclencheur.
```

---

### 4️⃣ ERREUR COURANTE

**Objectif** : Montrer pourquoi les solutions classiques échouent

**Caractéristiques** :
- Ce que la majorité fait mal
- Pourquoi ça ne fonctionne pas
- Amplification de la frustration

**Exemple** :
```
Plus tu culpabilises → plus tu stresses.
Plus tu stresses → plus tu ronges.

C'est un cercle invisible.

Et c'est pour ça que "juste arrêter" ne marche pas.
```

---

### 5️⃣ SOLUTION CONCRÈTE

**Objectif** : Positionner l'expertise métier

**Caractéristiques** :
- Adaptée au métier sélectionné
- Pratique et applicable
- Crédible et professionnelle
- Ancre l'autorité naturellement

**Exemple** :
```
En institut, on agit sur 2 niveaux :

1️⃣ Renforcer l'ongle (gainage / semi renforcé)
2️⃣ Créer une barrière physique qui casse le réflexe

Quand l'ongle est solide, lisse, esthétique…
Le geste devient moins automatique.
```

---

### 6️⃣ PROJECTION ÉMOTIONNELLE

**Objectif** : Créer le désir du résultat

**Caractéristiques** :
- Avant / Après émotionnel
- Bénéfice visible ou ressenti
- Visualisation du changement

**Exemple** :
```
Imagine :

Regarder tes mains sans honte.
Arrêter de les cacher.
Te sentir soignée.

C'est souvent là que le déclic se fait.
```

---

### 7️⃣ CTA STRATÉGIQUE

**Objectif** : Faciliter l'action (jamais forcer)

**Adaptation selon l'objectif** :
- **Éducation** → Sauvegarde / Partage
- **Visibilité** → Commentaire
- **Conversion** → RDV / DM / Lien

**Caractéristiques** :
- Naturel et non agressif
- Action claire et simple
- Suit logiquement le contenu

**Exemple** :
```
Si tu veux arrêter de te ronger les ongles,
ne commence pas par te juger.

Commence par te faire accompagner.

📍 RDV en bio
💬 Ou DM "ONGLES"
```

## Déclencheurs psychologiques intégrés

Chaque script intègre automatiquement :

✓ **Curiosity gap** - Question ouverte qui demande réponse
✓ **Identification** - Situation vécue par l'audience
✓ **Autorité implicite** - Expertise démontrée (pas affichée)
✓ **Projection du résultat** - Visualisation du bénéfice
✓ **Empathie profonde** - Compréhension du problème

## Adaptation par plateforme

### Instagram
- Phrases courtes et rythmées
- Visuels forts
- Storytelling émotionnel

### LinkedIn
- Ton plus structuré
- Expertise mise en avant
- Approche professionnelle

### Reel / TikTok
- Script oral fluide
- Timing précis (0-3s hook crucial)
- Rythme rapide

### Carrousel
- 1 idée forte par slide
- Progression logique
- Visuels alignés avec le texte

## Vérification d'alignement

Le système vérifie automatiquement :

1. **Cohérence thématique** - Le sujet ne dérive pas
2. **Présence des mots-clés** du thème source
3. **Absence de sujets génériques** non liés
4. **Alignement métier** - Solutions adaptées

## Utilisation dans Belleya

### 1. Boîte à idées
- Génère 5 idées stratégiques adaptées au métier
- Chaque idée inclut un script complet
- Vérification automatique de l'alignement

### 2. Studio de contenu
- Génération de script détaillé pour chaque contenu
- Adaptation automatique au format (post, reel, carrousel)
- Légende longue + version courte + hashtags

### 3. Calendrier éditorial
- Suggestion de thèmes selon piliers éditoriaux
- Scripts prêts à adapter avec visuels
- CTA adapté à l'objectif du mois

## Conseils d'utilisation

### Pour les utilisateurs

1. **Soyez précis sur le thème** - Plus c'est spécifique, plus c'est puissant
2. **Choisissez le bon objectif** - Éduquer / Attirer / Convertir / Fidéliser
3. **Adaptez à votre ton** - Le script est une base, personnalisez-le
4. **Testez plusieurs versions** - Générez 5 idées, choisissez la meilleure

### Pour les développeurs

Le système est dans `src/lib/contentAIGenerator.ts` :

**Fonctions clés** :
- `generateContentAI()` - Point d'entrée principal
- `generateDetailedScript()` - Génère le script complet 7 étapes
- `generateStrategicHook()` - Crée le hook pattern interrupt
- `generateConcreteSolution()` - Adapte la solution au métier
- `verifyTopicCoherence()` - Vérifie l'alignement thématique
