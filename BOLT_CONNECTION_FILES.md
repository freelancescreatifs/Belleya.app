## 🚀 Par où commencer ?

### Option A : Vous voulez aller vite
👉 Lisez `QUICK_CONNECT.md` et copiez-collez les commandes.

### Option B : Vous voulez un script automatique
👉 Exécutez `connect-to-bolt.sh` (ou `.bat` sur Windows).

### Option C : Vous voulez tout comprendre
👉 Lisez `CONNECT_BOLT_GUIDE.md` pour tous les détails.

### Option D : Vous configurez l'équipe
👉 Lisez `TEAM_SETUP_GUIDE.md` pour la configuration complète.
